• Polymorphism means many shapes"
  • Polymorphism can be expressed as 'One interface multiple functions'
  • Polymorphism can be static or dynamic
    • static: ran at compile time, aka early binding or static binding
    • dynamic: ran at run time
    • One variable can point to different types of objects
    • handy when you need to assign a group of objects to an array and then invoke each of their methods.
  • Objects can behave differently depending on thier type

1. Question: What does the following statement do? using System;
    • Imports the System.dll assembly AND allows types in that assembly to be referenced without specifying the namespace.
    • Imports the System.dll assembly, so that types in that assembly can be used.
    • Allows types in the System namespace to be referenced without specifying the namespace explicitly.

    Answer:

    Allows types in the System namespace to be referenced without specifying the namespace explicitly.
